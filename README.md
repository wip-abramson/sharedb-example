# ShareDb Example

Very basic example using a ShareDb instance with a React frontend.

## Configure MongoBb 

https://medium.com/swlh/get-up-and-running-with-mongodb-in-under-5-minutes-abc8770b1ef8

## Run server

* npm install
* npm start

## Run client:

* cd client
* npm install
* npm start
